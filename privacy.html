Politique de Confidentialité

1. Introduction

Kesiara Golden (ci-après "nous" ou "l'entreprise") s'engage à protéger votre vie privée. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos données personnelles.

2. Données Collectées

Nous collectons les informations suivantes:

  • Informations de commande (panier, quantités, prix)
  • Données de contact (numéro WhatsApp utilisé)
  • Données de navigation (localStorage du navigateur)

3. Utilisation des Données

Vos données sont utilisées pour:

  • Traiter vos commandes
  • Vous contacter via WhatsApp
  • Améliorer notre site web
  • Respecter les obligations légales

4. Sécurité des Données

Nous protégeons vos données avec des mesures de sécurité appropriées. Les données du panier sont stockées localement dans votre navigateur et ne sont jamais envoyées à des serveurs sans votre consentement.

5. Cookies

Notre site utilise le localStorage pour mémoriser votre panier. Vous pouvez le désactiver dans les paramètres de votre navigateur.

6. Droits de l'Utilisateur

Vous avez le droit de:

  • Accéder à vos données
  • Rectifier vos données
  • Supprimer vos données
  • Vous opposer au traitement
